Blaise Stats is an interactive web app focused on bringing new, innovative baseball stats to the public. Currently, Blaise Stats focuses on Late Game Slugging (LGS) and Early Game Slugging (EGS). LGS is a metric that depicts a player's Slugging percentage in the 8th inning and above; while, EGS is a metric that depicts a player's Slugging percentage in the 7th inning and below. These metrics can be used to visualize how a batter's efficiency at the plate changes as the game progresses. Note that Blasie Stats currently uses the Statcast data from the 2016, 2017, and 2018 MLB seasons. Therefore, players that did not play during these seasons will not be able to be searched. Moreover, some players' names caused issues when querering the data. A full list of 2016-1018 players that are not searchable by Blaise Stats can be found here.
